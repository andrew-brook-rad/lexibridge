### SESSION 1 - INITIALIZATION
Date: 2024-12-18

Project initialized for translation correction task.

Files configured:
- feature_list.json: 66 books to correct (gen through rev)
- app_spec.txt: Task specification for translation corrections
- coding_prompt.md: Agent workflow instructions

Input files:
- tools/interlinear/output/{book}-issues.txt - Lists flagged verses needing correction
- tools/interlinear/output/{book}.tsv - TSV files to edit

Issue types to fix:
1. Consecutive duplicate glosses (e.g., "bring bring")
2. German separable verb prefixes (e.g., "hervor" -> FORTH)
3. Unknown words in brackets (e.g., [rühmt] -> TOUCHES)
4. Death idiom errors ("des Todes sterben" -> OF-THE DEATH DIE)
5. Compound place names with brackets

Next steps:
- Start with Genesis (gen)
- Read gen-issues.txt to identify all flagged verses
- Correct each flagged verse in gen.tsv
- Run python3 scripts/consolidate_interlinear.py after corrections
- Mark gen as passing in feature_list.json
